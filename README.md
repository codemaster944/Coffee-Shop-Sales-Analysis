☕ Coffee Shop Sales Analysis

📌 Project Overview

This project focuses on analyzing coffee shop sales data to uncover meaningful business insights. Using Python and data analysis techniques, the goal is to understand customer behavior, identify top-performing products, and optimize sales strategies.

🎯 Objectives
Analyze sales trends over time
Identify best-selling products
Understand customer purchasing patterns
Evaluate store performance
Generate actionable business insights

🛠️ Tools & Technologies
Python
Pandas – Data manipulation & cleaning
NumPy – Numerical operations
Matplotlib & Seaborn – Data visualization
Jupyter Notebook

📂 Dataset Description

The dataset contains transactional data from a coffee shop, including:

Order ID
Product Name & Category
Quantity Sold
Unit Price
Transaction Date & Time
🔍 Key Steps Performed

1️⃣ Data Cleaning

Handled missing values
Converted data types (date/time)
Removed duplicates
Prepared dataset for analysis

2️⃣ Exploratory Data Analysis (EDA)

📅 Time-Based Analysis
Sales by hour, day, and month
Identified peak business hours

☕ Product Analysis
Top-selling products
Revenue by category

🏪 Store Performance
Comparison across locations
Best-performing store identification

💰 Revenue Analysis
Total revenue
Total orders
Average order value (AOV)

📊 Key Insights
Peak sales occur during specific hours of the day
A few products contribute to the majority of revenue
Certain days show consistently higher sales
Customer purchasing behavior follows clear patterns

📈 Visualizations

The project includes multiple visualizations such as:

Bar charts

Heatmaps

These help in better understanding trends and patterns.

💼 Business Impact

This analysis can help businesses:

Optimize staffing during peak hours

Increase overall profitability
